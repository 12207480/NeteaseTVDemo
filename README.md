# NeteaseTVDemo - 网易云音乐 tvOS 客户端

代码完全开源（代码质量粗糙，主打一个能跑就行，非常欢迎 Review），支持以下功能：

- 二维码登录
- 每日推荐（歌单、单曲）
- 歌单、播客、MV 分类
- 音视频播放器
- 歌词滚动
- 最近播放（单曲 声音 歌单 专辑 播客）
- 我的收藏（歌单 专辑 播客）
- 我的歌单
- 搜索

## 支持平台
Apple TV
